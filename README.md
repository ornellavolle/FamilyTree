# 🌳 Arbre Généalogique · Famille Piponnier

Application interactive pour explorer l'arbre généalogique de la famille Piponnier.

## Voir l'application en ligne

👉 **[Lien Streamlit à venir après déploiement]**

## Lancer en local

```bash
pip install streamlit plotly
streamlit run app.py
```

## Déployer sur Streamlit Cloud (gratuit)

1. Crée un compte sur [github.com](https://github.com)
2. Crée un nouveau dépôt public, uploade `app.py` et `requirements.txt`
3. Va sur [share.streamlit.io](https://share.streamlit.io)
4. Connecte ton GitHub, sélectionne le dépôt → **Deploy**
5. Tu reçois un lien du style `https://ton-nom-famille-piponnier.streamlit.app`

## Fichiers

- `app.py` — l'application principale
- `requirements.txt` — les dépendances Python
